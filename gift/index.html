FAQ

装不上看这里

装在哪里？要不要每个 vault 都装一遍？

全局 Skill，装一次在哪都能用。路径是 ~/.claude/skills/recording/（Mac）或 %USERPROFILE%\.claude\skills\recording\（Windows）——在用户家目录下，不管你在哪个项目、哪个 vault 里打开 Claude Code，这个 Skill 都能识别。

Claude Code 不认识"录音"命令？

三种可能：

1. Skill 没装好——确认 ~/.claude/skills/recording/SKILL.md 文件存在。没有的话重跑 bash install.sh。

2. Claude Code 没重启——退出重开一次。

3. 触发词要精确——试试直接输入 录音（两个字）或 录音，测试。

录音文件存在哪？

存到你当前 Claude Code 打开的目录下的 recordings/ 子文件夹——在哪个项目录的，就存哪个项目里，不会乱跑到别的地方。

转写文本放在同目录的 .md 文件里，包含：完整转写、要点提炼、下一步行动、洞察。

可以录多长？

单次建议 60 分钟以内（超长会分段处理）。一次 1 小时客户会、一场团队会议都 OK。
